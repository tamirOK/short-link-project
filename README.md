# Short Url Project

Short link project is simple tool for shortening long and ugly links. You provide url and then application generates
new shorter one. Also you can provide your own shortened version of initial url address.


### Installing

```
pip install -r requirements.txt
```

## Running

To run application on your machine

```
python manage.py runserver
```

## Built With

* [Django](https://www.djangoproject.com/) - The web framework used
